## AUTOMATION RULES

### Comment Detection Settings
- **Exact match:** OFF (allows "VISION please" to trigger)
- **Case sensitive:** OFF
- **Include replies:** ON
- **Exclude own comments:** ON

### Rate Limiting
- **Same user cooldown:** 24 hours (prevents spam triggers)
- **Max DMs per hour:** Follow ManyChat limits

### Error Handling
- If DM fails ‚Üí Add to "Failed DM" list for manual follow-up
- If user has DMs closed ‚Üí Reply to comment: "Hey! Your DMs are closed ‚Äî turn them on and comment again to get the guide! üî•"
