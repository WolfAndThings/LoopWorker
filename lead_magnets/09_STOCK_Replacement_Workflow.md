## THE REPLACEMENT WORKFLOW

### Step 1: Identify What You Need
Common business image needs:
- Hero images for landing pages
- Blog post headers
- Social media content
- Presentation backgrounds

## REPLACEMENT PROMPTS BY CATEGORY

### Hero/Landing Page Images

**Stock:** Generic handshake, laptop lifestyle shot

**AI Prompt:**
```
Professional business concept image. [Your brand's lighting style], [your color palette]. Abstract representation of [concept: growth/connection/innovation]. Clean, modern, ample space for text overlay. No faces, no hands. Sophisticated corporate aesthetic. [Your preferred ratio].
```

**Example:**
```
Professional business concept image. Soft, diffused lighting, blue and white color palette with gold accents. Abstract representation of growth - geometric shapes rising, elegant lines suggesting upward movement. Clean, modern, ample space for text on left third. No faces, no hands. Sophisticated corporate aesthetic. 16:9 ratio.
```

---

### Blog Post Headers

**Stock:** Obvious metaphor (lightbulb = idea, mountain = challenge)

**AI Prompt:**
```
Blog header image for article about [TOPIC]. [Your brand's style] aesthetic. Subtle, sophisticated visualization of the concept. [Color palette]. Clean composition with space for title overlay. Avoid clichés. [Ratio, usually 16:9 or 2:1].
```

**Example for "Building Better Habits":**
```
Blog header image for article about building better habits. Minimal, clean aesthetic with soft morning light. Subtle visual of daily ritual - perhaps a perfectly made bed corner, or organized desk detail. Warm neutral tones, cream and soft brown. Clean composition with space for title on left. Avoid clichés like checklist or alarm clock. 16:9 ratio.
```

---

### Social Media Content

**Stock:** Overused templates, obvious graphics

**AI Prompt:**
```
Social media image for [PLATFORM]. Topic: [SUBJECT]. [Your brand style] - [lighting, colors, mood]. [Content type: quote background, announcement, product feature]. Space for text overlay. [Platform-specific ratio].
```

**Example for Instagram:**
```
Instagram post background. Topic: motivation. Minimal aesthetic - soft, dreamy gradient in brand colors (sage green to cream). Abstract, calming, with visible texture. Optimized for text overlay in center. No distracting elements. 1:1 ratio.
```

---

### Presentation Backgrounds

**Stock:** Boring gradients, overused patterns

**AI Prompt:**
```
Professional presentation background. [Your brand style]. [Color palette] as primary tones. Subtle, non-distracting pattern or gradient. Clean, corporate but not boring. Optimized for dark text overlay. 16:9 ratio for slide deck.
```

---

### Email Newsletter Headers

**Stock:** Seasonal/obvious themed images

**AI Prompt:**
```
Email newsletter header image. [Current theme/topic]. [Brand style] aesthetic with [color palette]. Clean, professional, engaging without being distracting. Optimized for email format. 600px wide concept, 3:1 ratio.
```

---

### Abstract Concept Images

**Stock:** Overused metaphors (puzzle pieces, gears, light bulbs)

**AI Prompt:**
```
Abstract concept visualization: [CONCEPT]. Sophisticated, minimal aesthetic. [Your brand colors]. No clichéd symbols. Instead, use [abstract shapes / flowing forms / architectural elements / natural textures]. Subtle, elevated, could work for a premium brand. [Ratio].
```

**Concept Alternatives:**
- Growth: Rising organic forms, architectural lines
- Connection: Interlocking shapes, subtle networks
- Innovation: Dynamic geometry, light refraction
- Trust: Solid forms, balanced composition
- Speed: Flowing lines, motion blur
- Quality: Precise details, premium materials

## BATCH CREATION SYSTEM

### Monthly Content Planning

**Week 1: Audit Needs**
- List all content pieces for the month
- Identify image needs for each
- Group by type (hero, social, blog, etc.)

**Week 2: Batch Generate**
- Create in focused sessions
- Generate 10 variations per concept
- Pick best 3, refine if needed

**Week 3: Organize Library**
- Sort by type and topic
- Name files descriptively
- Create easy-access folder structure

**Week 4: Deploy & Track**
- Use in content
- Note what performs best
- Inform next month's approach
