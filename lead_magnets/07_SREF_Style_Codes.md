# SREF STYLE CODE COLLECTION
## 15 Tested Midjourney Style References

---

## HOW TO USE SREF CODES

Add to the end of your Midjourney prompt:
```
/imagine [your prompt] --sref [code number]
```

Or combine with style weight:
```
/imagine [your prompt] --sref [code number] --sw 100
```

**Style Weight (--sw):**
- 0 = No style influence
- 100 = Default influence
- 1000 = Maximum style influence

## COMBINING CODES

You can blend multiple style references:

```
/imagine [prompt] --sref 483744325 2847261954 --sw 150
```

This blends the two styles together.

**Pro tip:** Start with one code, then add a second at lower influence to subtly modify.

## CREATING YOUR OWN STYLE CODES

### Method 1: From Image
Upload an image you love:
```
/imagine [upload image] --sref
```
Midjourney will extract the style as a code.

### Method 2: From Generation
If you get a result you love, run:
```
/describe [image]
```
Then use that description to inform future prompts.

### Method 3: Random Discovery
```
/imagine [prompt] --sref random
```
Midjourney picks a random style. Note ones you like.

## WORKFLOW

1. Pick a style code that matches your brand vibe
2. Write your content prompt
3. Add the SREF at the end
4. Generate 4 images
5. Pick best, iterate
6. Use same SREF for all brand images = consistency
